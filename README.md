# Slamdunk

Slamdunk is a React + TypeScript single-page app where fans can read and submit
testimonials about how *Slam Dunk* impacted their lives.

## Project brief

- Purpose: collect and showcase community stories in a clean, editorial interface
- Stack: React 19, TypeScript, Vite, React Router
- Visual direction: Ferrari-inspired black/white contrast with restrained red accents

## Run locally

```bash
npm install
npm run dev
```

## Available scripts

- `npm run dev` - start development server
- `npm run lint` - run ESLint
- `npm run build` - type-check and build production bundle
- `npm run preview` - preview production build locally
